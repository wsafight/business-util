> 那些在中文互联网上"普度众生"的慷慨平台

## 前端部署与存储类

### [七牛云](https://www.qiniu.com)
- **官网**: https://www.qiniu.com
- **免费额度**: 10GB 存储、10GB CDN 流量/月、100 万次 PUT/DELETE 请求
- **特点**: 老牌对象存储、CDN 加速、图片处理
- **适合**: 静态资源托管、图片存储、文件 CDN
- **菩萨指数**: ⭐⭐⭐⭐⭐（国内存储界的活菩萨）

### [又拍云](https://www.upyun.com)
- **官网**: https://www.upyun.com
- **免费额度**: 加入又拍云联盟可获得每月免费 CDN 流量（需在网站添加 Logo）
- **特点**: CDN 加速、云存储、云处理
- **适合**: 个人博客、小型网站
- **菩萨指数**: ⭐⭐⭐⭐⭐（良心联盟计划）

### [阿里云 OSS](https://www.aliyun.com/product/oss)
- **官网**: https://www.aliyun.com/product/oss
- **免费额度**: 新用户 6 个月试用、40GB 标准存储包
- **特点**: 阿里云生态、稳定可靠
- **适合**: 需要对象存储的应用
- **菩萨指数**: ⭐⭐⭐

### [腾讯云 CloudBase](https://cloudbase.net)
- **官网**: https://cloudbase.net
- **免费额度**: 5GB 云存储、5GB CDN 流量/月、2 万次云函数调用
- **特点**: 一站式后端云服务、微信小程序御用
- **适合**: 小程序、Web 应用
- **菩萨指数**: ⭐⭐⭐⭐

### [Vercel](https://vercel.com) （国内访问友好）
- **官网**: https://vercel.com
- **免费额度**: 同国外版，国内访问相对稳定
- **特点**: 虽是国外服务，但对国内开发者友好
- **适合**: 面向海外用户的前端项目
- **菩萨指数**: ⭐⭐⭐⭐

## 使用建议

1. **实名认证**: 国内大多数平台需要实名认证,提前准备好
2. **新人优惠**: 充分利用各大云平台的新用户试用额度
3. **备案要求**: 使用国内服务器需要 ICP 备案,境外服务器不需要
4. **流量监控**: 密切关注流量使用,避免超额产生费用
5. **多平台组合**: 合理搭配国内外服务,取长补短
6. **政策变化**: 国内平台政策变化较快,保持关注

## 特别说明

### 国内 vs 国外平台

**国内平台优势**:
- 访问速度快(国内用户)
- 客服支持好(中文沟通)
- 支付方便(支持支付宝/微信)

**国内平台劣势**:
- 需要实名认证
- 网站需要备案
- 免费额度相对较少
- 政策变化可能较快

**推荐策略**:
- 静态资源 CDN → 七牛云/又拍云
- 应用部署 → Vercel/Cloudflare Pages (国外) + Gitee Pages (国内)
- 对象存储 → 七牛云(小文件) + 阿里云 OSS(大项目)
- 数据库 → Supabase/PlanetScale (国外平台,国内可访问)
- API 调试 → Apifox
- 文档协作 → 语雀/飞书文档
- 认证服务 → Authing/小程序云开发
- AI 能力 → Kimi/通义千问

## 总结

国内的赛博菩萨虽然在免费额度上可能不如国外慷慨,但在本土化服务、访问速度和客户支持上有独特优势。特别是:

**真·大菩萨**:
- 语雀: 国内最良心的文档平台
- 七牛云/又拍云: 存储 CDN 的良心选择
- Gitee: 国内访问快的代码托管
- Apifox: 国产 API 工具的骄傲
- iconfont / iconpark: 图标资源的宝库
- DNSPod: DNS 解析的首选
- B站: 免费学习资源的天堂
- 飞书: 团队协作的典范
- 钉钉: 企业办公的好帮手
- Kimi: 长上下文 AI 的佼佼者

**建议**:
- 不要过度依赖单一平台
- 合理利用新用户优惠
- 项目成熟后支持这些平台
- 关注开源社区,回馈社会

愿这些国内赛博菩萨继续"普度众生",功德无量!
